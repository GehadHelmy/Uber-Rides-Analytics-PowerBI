# Uber Rides Analytics Dashboard

## Overview
This project is a Power BI dashboard designed to analyze Uber ride data and provide actionable business insights through interactive visualizations.

The dashboard focuses on key performance indicators such as revenue, customer behavior, ride demand patterns, vehicle usage, and location performance.

## Dashboard Pages

### 1. Home
- Executive summary of key metrics
- Quick overview of business performance

### 2. Peak Time Analysis
- Ride demand by time and date
- Identification of peak operating hours
- Trend analysis for ride frequency

### 3. Location Analysis
- Most active pickup and drop-off locations
- Geographic demand patterns
- Location performance comparison

### 4. Customer Analysis
- Customer behavior insights
- Ride distribution across customer segments
- Customer engagement metrics

### 5. Vehicle Analysis
- Vehicle utilization tracking
- Performance comparison between vehicle categories
- Operational efficiency insights

### 6. Revenue Analysis
- Revenue trends and growth
- Revenue distribution across different dimensions
- Business performance monitoring

## Tools & Technologies
- Microsoft Power BI
- Data Modeling
- DAX (Data Analysis Expressions)
- Data Visualization
- Business Intelligence Analytics

## Key Features
- Interactive filtering and slicing
- Dynamic KPI tracking
- Trend analysis
- Location-based insights
- Customer and vehicle performance monitoring
- Revenue reporting

## Purpose
The goal of this project is to transform raw Uber ride data into meaningful insights that support data-driven decision-making and improve operational efficiency.
